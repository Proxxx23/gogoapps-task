# Node.js Toolchain

## Installation

First, install Node.js on your machine: [How to install Node.js on MacOS](https://nodejs.org/tr/download/package-manager/#macos)

```bash
$ npm install
```

## Running app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

App will run on: http://127.0.0.1/3000

## Running tests

```bash
# unit tests
$ npm run test
```
